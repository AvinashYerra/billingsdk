## Plan Interface

The `Plan` interface is used to define the structure of a pricing plan and is shared across all the `PricingTable` components.

```tsx
interface Plan {
    id: string
    title: string
    description: string
    highlight?: boolean
    type?: 'monthly' | 'yearly'
    monthlyPrice: string
    yearlyPrice: string
    buttonText: string
    badge?: string
    features: {
        name: string
        icon: string
        iconColor?: string
    }[],
    benefits?: string[]
}
```

Whenever you add a new `PricingTable` component, a file `/lib/billing-sdk-const.ts` will be created with the `plans` array.

A sample of the `/lib/billing-sdk-const.ts` file is:

