## How to Change Themes

To change the theme of your components, simply modify the CSS variables in your `global.css` file. The current theme uses these variables:

```css title="src/app/global.css"
:root {
  --radius: 0.625rem;
  --background: oklch(1 0 0);
  --foreground: oklch(0.145 0 0);
  --card: oklch(1 0 0);
  --card-foreground: oklch(0.145 0 0);
  --popover: oklch(1 0 0);
  --popover-foreground: oklch(0.145 0 0);
  --primary: oklch(0.205 0 0);
  --primary-foreground: oklch(0.985 0 0);
  --secondary: oklch(0.97 0 0);
  --secondary-foreground: oklch(0.205 0 0);
  --muted: oklch(0.97 0 0);
  --muted-foreground: oklch(0.556 0 0);
  --accent: oklch(0.97 0 0);
  --accent-foreground: oklch(0.205 0 0);
  --destructive: oklch(0.577 0.245 27.325);
  --border: oklch(0.922 0 0);
  --input: oklch(0.922 0 0);
  --ring: oklch(0.708 0 0);
}

.dark {
  --background: oklch(0.145 0 0);
```

## Color Format

We use OKLCH color format:
- **L (Lightness)**: 0-1 (0 = black, 1 = white)
- **C (Chroma)**: 0-0.4+ (saturation level)
- **H (Hue)**: 0-360 (color wheel degrees)

Examples:
- Blue: `oklch(0.6 0.25 253)`
- Green: `oklch(0.5 0.2 142)`
- Orange: `oklch(0.6 0.2 42)`
- Red: `oklch(0.55 0.25 12)`

## Key Variables

- `--primary`: Main brand color
- `--secondary`: Secondary elements
- `--accent`: Highlighted elements
- `--muted`: Subtle backgrounds
- `--border`: Border colors
- `--ring`: Focus rings
