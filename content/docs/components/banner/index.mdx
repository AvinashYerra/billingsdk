## Usage

```tsx
import { Banner } from "@/components/billingsdk/banner";
```


```tsx
// Default Banner
<Banner
  title="üéâ Start your free trial today!"
  description="Get 30 days free access to all premium features"
  buttonText="Start Free Trial"
  buttonIcon={<Rocket />}
  buttonLink="https://example.com/signup"
  variant="default" // default, minimal, popup
/>
```

```tsx
// Minimal Banner
<Banner
  title="üéâ Start your free trial today!"
  description="Get 30 days free access to all premium features"
  variant="minimal" // default, minimal, popup
/>
```

```tsx
// Popup Banner
<Banner
  title="üéâ Start your free trial today!"

  variant="popup" // default, minimal, popup

```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `className` | `string` | ‚ùå | Additional CSS classes for styling |
| `title` | `string` | ‚úÖ | The title of the banner |
| `description` | `string` | ‚ùå | The description of the banner |
| `buttonText` | `string` | ‚ùå | The text of the button |
| `buttonIcon` | `React.ReactNode` | ‚ùå | The icon of the button |
| `buttonLink` | `string` | ‚ùå | The link of the button |
| `variant` | `"default" | "minimal" | "popup"` | ‚úÖ | The variant of the banner |
| `autoDismiss` | `number` | ‚ùå | The time in milliseconds to auto dismiss the banner |
| `onDismiss` | `() => void` | ‚ùå | The function to call when the banner is dismissed |
