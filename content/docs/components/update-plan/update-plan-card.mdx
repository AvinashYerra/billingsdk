## Usage

```tsx
import { UpdatePlanCard } from "@/components/billingsdk/update-plan-card";
import { plans } from "@/lib/const";

<UpdatePlanCard
  currentPlan={plans[0]}
  plans={plans}
  onPlanChange={(planId) => {
    // Handle plan change
    console.log('Plan changed to:', planId);
  }}
  title="Upgrade Your Plan"
/>
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `currentPlan` | `Plan` | ✅ | The user's current plan object |
| `plans` | `Plan[]` | ✅ | Array of available plans to choose from |
| `onPlanChange` | `(planId: string) => void` | ✅ | Callback function when a plan is selected |
| `className` | `string` | ❌ | Additional CSS classes for styling |
| `title` | `string` | ❌ | Custom title for the card (default: "Upgrade Plan") |

## Features

- **Compact Design**: Card-based layout perfect for limited spaces
- **Plan Comparison**: Visual comparison of available plans
- **Monthly/Yearly Toggle**: Switch between billing cycles
- **Interactive Selection**: Radio button selection with hover effects
- **Smooth Animations**: Motion animations for plan transitions
- **Responsive Layout**: Adapts to different screen sizes
- **Feature Display**: Shows plan features in a clean format

## Use Cases

- **Dashboard Widgets**: Embed in user dashboards
- **Sidebar Components**: Display in navigation sidebars
- **Profile Pages**: Show in user profile sections
- **Settings Panels**: Include in account settings
- **Modal Content**: Use as content in larger modals

## Design Features

The component features:
- Clean card design with shadows
- Gradient backgrounds for selected plans
- Smooth hover transitions
- Consistent spacing and typography
- Primary color accents for selection states
- Responsive grid layout
