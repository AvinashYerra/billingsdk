## Overview

The Pricing Table Three component provides a third design option for pricing displays.

## Usage

```tsx
import { PricingTableThree } from "@/components/billingsdk/pricing-table-three";
import { plans } from "@/lib/const";

<PricingTableThree 
  plans={plans}
  title="Pricing Plans"
  description="Find the right plan for your business"
  onPlanSelect={(planId) => console.log('Selected plan:', planId)}
/>
```
